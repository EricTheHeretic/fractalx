# FractalX

**Hide secret messages inside fractals.**

Send the image over the internet.  
Send the password to friend.  
Only the recipient with the password can read the message encrpyted in your image.

---

### How it works

1. Type your message + create a password
2. Click "Generate Fractal + Hide Message"
3. Download the PNG and send it anywhere (X, email, Discord, etc.)
4. Send the password to the recipient.
5. Recipient opens this app, uploads the PNG, enters the password, and reads your message

Each image has its own password. The fractal itself is the mathematical carrier of the message.

---

### Features

- Real fractal generation (Mandelbrot, Julia, Burning Ship)
- AES encryption + LSB steganography
- Message hidden directly inside the pixels
- Simple and beginner-friendly interface
- Beautiful, shareable fractal images

---

### Quick Start

1. Type your message
2. Create a password
3. Click the big button
4. Download the PNG and send it
5. Send the password via physical mail

To decode: Upload the PNG + enter the password.
